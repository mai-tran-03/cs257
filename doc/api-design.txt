REQUEST: api/countries
    GET parameters:
        attribute (Optional, default: '') -- return all countries' data about
            the specified attributes. The user can query as many flag 
            attributes as wanted. They can choose at most one continent to
            query. 

RESPONSE: A JSON list of dictionaries, each representing a country. They are 
        ordered alphabetically by country name. Each dictionary will have the
        following fields:
        name           -- (TEXT) name of the country
        iso3           -- (TEXT) ISO 3166-1 alpha-3, three letter standardized code to represent countries 
        tld            -- (TEXT) ISO 3166-1 alpha-2, two letter standardized international code to represent countries 
        all attributes inputted with their counts/values (INTEGER or BOOLEAN)
        continent name -- (TEXT) only if any search continent was specified


EXAMPLE:
    api/countries?saltires=1&blue=1&circles=1

    [
      {
        "blue": false,
        "circles": 1,
        "country_name": "Afghanistan",
        "iso3": "AFG",
        "saltires": 0,
        "tld": "af"
      },
      {
        "blue": false,
        "circles": 0,
        "country_name": "Albania",
        "iso3": "ALB",
        "saltires": 0,
        "tld": "al"
      },

    ...

      {
        "blue": false,
        "circles": 0,
        "country_name": "Zimbabwe",
        "iso3": "ZWE",
        "saltires": 0,
        "tld": "zw"
      }
    ]




REQUEST: api/country/{name}

RESPONSE: A JSON dictionary with all the attributes of the country. The name is
case insensitve. The following attributes returned are: 
    tld             -- (TEXT)  ISO 3166-1 alpha-2, two letter standardized international code to represent countries
    iso3            -- (TEXT) ISO 3166-1 alpha-3, three letter standardized code to represent countries
    name            -- (TEXT) Name of the selected country
    other_names     -- (TEXT) Other names of the selected country, e.g., official or 
                        local. Returns NULL if none stored
    area            -- (INTEGER) Total area of the country, including land and water, in sq km
    population      -- (INTEGER) Total population of the country established in 2024
    continent_name  -- (TEXT) Africa, Asia, Europe, North America, Oceania, or South America
    bars            -- (INTEGER) Number of vertical bars on the flag
    stripes         -- (INTEGER) Number of horizontal stripes on the flag
    bends           -- (INTEGER) Number of diagonal stripes on the flag, 
    red             -- (BOOLEAN) true if red is present on the flag, false otherwise
    green           -- (BOOLEAN) true if green is present on the flag, false otherwise
    blue            -- (BOOLEAN) true if blue is present on the flag, false otherwise
    gold_yellow     -- (BOOLEAN) true if gold or yellow is present on the flag, false otherwise
    white           -- (BOOLEAN) true if white is present on the flag, false otherwise
    black_grey      -- (BOOLEAN) true if black or grey is present on the flag, false otherwise
    orange_brown    -- (BOOLEAN) true if orange or brown is present on the flag, false otherwise
    pink_purple     -- (BOOLEAN) true if pink or purple is present on the flag, false otherwise
    main_hue        -- (TEXT) Predominant color on the flag (tie-breaks from the 
                        topmost, to most central, and then to leftmost hue)
    circles         -- (INTEGER) Number of circles on the flag
    crosses         -- (INTEGER) Number of (upright) crosses
    saltires        -- (INTEGER) Number of diagonal crosses
    quarters        -- (INTEGER) Number of quartered sections
    sun_stars       -- (INTEGER) Number of sun/star symbols
    crescent_moon   -- (BOOLEAN) true if any crescent moon symbol present, false otherwise
    triangle        -- (BOOLEAN) true if any triangle present, false otherwise
    inanimate_image -- (BOOLEAN) true if any inanimate image present, false otherwise
    animate_image   -- (BOOLEAN) true if any animate image present, false otherwise
    text            -- (BOOLEAN) true if any letter or writing present, false otherwise
    crest_emblem    -- (BOOLEAN) true if any emblem or crest present, false otherwise
    border          -- (BOOLEAN) true if any rectangular border present, false otherwise
    trapezoid       -- (BOOLEAN) true if any trapezoid present, false otherwise
    languages       -- (BOOLEAN) list of strings of most popular languages spoken
