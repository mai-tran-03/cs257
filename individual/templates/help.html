Country Flags API: endpoint reference
All responses are in JSON

REQUEST: /
RESPONSE: A greeting from this API to the user.

REQUEST: /help
RESPONSE: Current message.

REQUEST: /colors[?red={1}&green={1}&blue={1}&
                gold={1}&white={1}&black={1}&orange={1}]
RESPONSE:
    A list of country names whose flags contain all the requested colors.
    If specific color(s) are provided as GET parameters, e.g.,
        /colors?red=1&green=1&blue=1&gold=1&white=1&black=1&orange
    will return ['Belize', 'Venezuela'].
    
    If no color parameters are provided, i.e.,
        /colors
    will return a dictionary where each color maps to a list of countries
    whose flags contain that color, i.e., {color: [list of country names], ...}.
